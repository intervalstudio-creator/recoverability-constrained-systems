# Recoverability Boundary Engine v4

This package includes the promised:
- multi-case selector (NHS + offshore + legal + identity)
- timeline simulation
- visual boundary diagram
- logs + audit output

## Deploy on GitHub Pages
1. Create a GitHub repository, for example: `recoverability-demo`
2. Upload `index.html`
3. Go to **Settings** -> **Pages**
4. Under **Build and deployment**, choose **Deploy from a branch**
5. Select branch `main` and folder `/root`
6. Save

Your live URL will be:

`https://YOUR-USERNAME.github.io/recoverability-demo/`

## Suggested live usage
1. Start with the NHS case
2. Show the timeline crossing the recoverability window
3. Show the visual boundary diagram
4. Open the logs and audit table
5. Change one condition live to show where admissibility collapses
