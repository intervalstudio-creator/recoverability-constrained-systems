System invariant
A message may propagate only while its validity remains recoverable in time under real conditions.
If it cannot be verified, challenged, or corrected in time:
-> propagation is non-admissible

Merged execution layer
RCCS now explicitly operates in three equivalent execution forms:
- Human execution form
- Paper continuity form
- Offline device runtime form

Non-standalone note
Operational use must preserve the invariant across identity, message state, challenge, propagation,
resolution, acknowledgment, relay, fallback, and correction reachability.
