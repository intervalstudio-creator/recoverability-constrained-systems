# Human, Paper, and Device Forms

## Human communication layer
Voice, gesture, and relay communication are admissible only while claim state, challengeability, and correction remain recoverable in time under real conditions.

## Paper continuity layer
Paper forms are admissible only if message state, challenge state, and correction state remain visible, attributable, and recoverable.

## Offline device runtime
Peer-to-peer and store-and-forward communication are admissible only if message validity remains recoverable despite infrastructure absence, delay, and synchronization loss.

## Equivalence requirement
All forms must preserve identical admissibility outcomes.
